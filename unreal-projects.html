Unreal Engine Projects

Unreal Engine 5 Projects

Advanced Movement System

A comprehensive character movement pack leveraging Unreal Engine's Blueprint system and animation morphing technology for seamless character movement.

Key Features

  • Dynamic Running Mechanics: Speed adjustments based on terrain and player input.
  • Smooth Jumping System: Air control allowing players to adjust trajectory mid-air.
  • Fluid Sliding Mechanics: Triggered by user input with friction and momentum physics.
  • Advanced Mantling System: Enables smooth obstacle traversal by dynamically detecting ledges.
  • Realistic Climbing Mechanics: Uses IK (Inverse Kinematics) for precise character placement on surfaces.

Technical Implementation

  • Blueprint Scripting: Fully implemented using Unreal Engine's visual scripting system for rapid iteration and prototyping.
  • Animation Morphing: Seamless state transitions between animations using Unreal Engine's Animation Blueprint.
  • Physics Integration: Adjusted player momentum and collision handling for realistic movement.
  • Debugging Tools: Used Unreal Engine's debug mode for fine-tuning physics values and animation timing.

UEFN Projects

A collection of custom-designed maps that showcase my technical expertise in Unreal Engine for Fortnite (UEFN). Each project demonstrates my ability to integrate advanced mechanics, optimize gameplay, and create immersive experiences.

Horror Map

Step into a chilling world where tension and terror await at every corner. This map combines environmental storytelling with trigger-based events for a suspenseful horror experience.

Key Features:
  • Dynamic Lighting System: Created with Unreal Engine's Niagara particle system and Blueprint scripting for real-time lighting changes triggered by player actions.
  • Trigger-Based Events: Scripting via UEFN Verse to activate key gameplay sequences based on player movement and decisions.
  • Atmospheric Audio Integration: Leveraged Unreal's audio system for directional sound cues and dynamic ambient effects, enhancing immersion.

The Bridges (1v1)

A competitive one-on-one battle across dynamically shifting bridges. Players must outwit their opponent through mastery of combat and navigation.

Key Features:
  • Custom Scoring System: Built with UEFN Verse, tracks kills and objectives for balanced competition.
  • Dynamic Bridge Mechanics: Utilized physics-based animations and timed triggers to create bridges that shift and challenge players.
  • Optimized for Competitive Play: Extensive playtesting to ensure balanced gameplay and minimal lag.

The Bridges (4v4v4v4)

A multi-team map designed for strategic coordination and intense gameplay.

Key Features:
  • Multi-Team Management System: Implemented via UEFN Verse, supports four distinct teams with synchronized objectives.
  • Advanced Spawn Logic: Optimized spawn locations using custom Blueprint logic to reduce spawn camping.
  • Team-Based Scoring Mechanics: Dynamic scoring system rewarding teamwork and strategy.

Fall Guys-Inspired Map

A physics-based obstacle course inspired by the popular game "Fall Guys," challenging players to reach the finish line.

Key Features:
  • Physics-Based Obstacles: Designed and tested using Unreal's physics engine to create interactive challenges like moving platforms and swinging barriers.
  • Checkpoint System: Custom Blueprint logic for player progression, allowing restarts without losing overall progress.
  • Time Trial Mechanics: Integrated leaderboard and timer system to encourage competitive play.
